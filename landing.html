Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as one gets closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matpotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudieness, windspeed, and humidity. This site provides the source data and vizualizations created as part of the analysis, as well as explainations and descriptions of any trends and correlations witnessed.
